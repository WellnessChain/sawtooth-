Hyperledger Sawtooth
-------------

Hyperledger Sawtooth是一个建设的企业解决方案,部署和
运行分布式分类帐(也称为blockchains)。它提供了一个非常
模块化和灵活的平台实现基于事务的更新
共享状态不可信方之间协调一致通过算法。
